MPI негіздері

Message Passing Interface бағдарламалаудың бастапқы қадамдары

MPI - бұл параллельдік бағдарламалау үшін кең таралған стандарт. Ол процессер арасында хабар алмасуды ұйымдастырады және жоғары өнімді есептеулердің негізгі құралы болып табылады.

Басты / MPI негіздері
MPI деген не?

MPI (Message Passing Interface) - бұл 1990-шы жылдардың басында дамытылған параллельдік бағдарламалау стандарты. Ол:

Хабар алмасу моделі

Процесстер бір-бірімен хабарлар арқылы қарым-қатынас жасайды

MIMD архитектурасы

Әр процессор өз деректерімен және инструкцияларымен жұмыс істей алады

Масштабтанатын

Ондаған мың процессорларға дейін өсіру мүмкіндігі бар

Негізгі тұжырымдамалар

Процесс рангі

Әр процестің бірегше идентификаторы (0-ден басталады)

int rank;
MPI_Comm_rank(MPI_COMM_WORLD, &rank);

Процессер саны

Коммуникатордағы барлық процессердің саны

int size;
MPI_Comm_size(MPI_COMM_WORLD, &size);

Коммуникатор

Процесстердің коммуникациялық тобы (MPI_COMM_WORLD)

MPI_Comm comm = MPI_COMM_WORLD;
Негізгі функциялар
Функция Сипаттама Мысал
MPI_Init MPI ортасын бастау MPI_Init(&argc, &argv);
MPI_Finalize MPI ортасын аяқтау MPI_Finalize();
MPI_Comm_rank Ағымдағы процестің рангін алу MPI_Comm_rank(comm, &rank);
MPI_Comm_size Процессер санын алу MPI_Comm_size(comm, &size);
MPI_Send Хабар жіберу MPI_Send(&data, count, type, dest, tag, comm);
MPI_Recv Хабар қабылдау MPI_Recv(&data, count, type, source, tag, comm, &status);

Толық мысал бағдарлама
mpi_complete_example.c 
#include "mpi.h"
#include <stdio.h>
#include <string.h>

int main(int argc, char *argv[]) {
    int rank, size;
    char processor_name[MPI_MAX_PROCESSOR_NAME];
    int name_len;
    
    // MPI бастау
    MPI_Init(&argc, &argv);
    
    // Процесс туралы ақпарат алу
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    MPI_Get_processor_name(processor_name, &name_len);
    
    printf("==============================\n");
    printf("Процесс: %d / %d\n", rank, size);
    printf("Процессор: %s\n", processor_name);
    printf("==============================\n");
    
    // Хабар алмасу мысалы
    if (size >= 2) {
        if (rank == 0) {
            // 0 процесі 1 процесіне хабар жібереді
            char message[] = "Сәлем, MPI әлемі!";
            MPI_Send(message, strlen(message)+1, MPI_CHAR, 1, 0, MPI_COMM_WORLD);
            printf("0 процесі хабар жіберді: %s\n", message);
        } 
        else if (rank == 1) {
            // 1 процесі хабарды қабылдайды
            char message[100];
            MPI_Status status;
            MPI_Recv(message, 100, MPI_CHAR, 0, 0, MPI_COMM_WORLD, &status);
            printf("1 процесі хабар қабылдады: %s\n", message);
        }
    }
    
    // MPI аяқтау
    MPI_Finalize();
    
    return 0;
}
