АТУ MPI Параллельдік Есептеулер Орталығы

Message Passing Interface - Жоғары Өнімді Есептеулер

Алматы Технологиялық Университетінің MPI технологиялары бойынша оқу-әдістемелік кешені. Параллельдік бағдарламалаудың негізгі стандартымен жұмыс істеуді үйреніңіз.

Бастау Практика 
#include "mpi.h"
int main(int argc, char *argv[]) {
    MPI_Init(&argc, &argv);
    int rank, size;
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    printf("Process %d of %d\n", rank, size);
    MPI_Finalize();
    return 0;
}

АТУ-да MPI жылдам бастау

Алматы Технологиялық Университетінің оқу ортасында MPI бағдарламалауды үйрену

atu_mpi_hello.c 
#include "mpi.h"
#include <stdio.h>

int main(int argc, char *argv[]) {
    int rank, size;
    
    // MPI бастау
    MPI_Init(&argc, &argv);
    
    // Ағымдағы процестің рангін алу
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    
    // Барлық процессер санын алу
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    
    printf("АТУ MPI: Сәлем! Мен %d процесім, барлығы %d процесс\n", rank, size);
    
    // MPI аяқтау
    MPI_Finalize();
    
    return 0;
}
